# 📥 動画ダウンロードスクリプト

`requests` と `tqdm` を使って、ダウンロードの進行状況も表示してくれます

---

## 🚀 機能

✨ **ストリーミングダウンロード**で  
　　大きなファイルも効率的に扱えます

📊 **進捗バー**で  
　　ダウンロード状況が一目でわかります

🌐 **カスタムHTTPヘッダー**で  
　　サイトに応じた柔軟なアクセスが可能です

🛡️ **個人情報を含まない**  
　　安全なコード設計になっています

---

## 🧠 必要なライブラリ

以下をインストールしてください💡

```bash
pip install requests tqdm
